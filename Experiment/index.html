Badvision Keyboard Experiment

Hello and thank you for participating in the Badvision keyboard experiment!

The goal of this experiment is to compare how well you type using different types of keyboards and also to test out a protype typing hint system.

Please note this requires a tablet and will not work with a mouse!

None of the data collected will be published or shared with any parties, this is only for a graded class project.

There are a total of 4 different trials, but the whole experiment should not take more than 15 minutes in total. Please try to participate in a quiet location without distraction if possible. Between each of the 4 sections, you can pause for as long as you'd like and come back to it and complete later.

Experiment instructions

In this experiment you type sentences that appear at the top of the screen.

You do not need to correct errors. Just keep typing.

If you make a mistake, start from the first white letter in the sentence. Letters you already typed turn blue.

There are 4 trials with different keyboards. At the beginning of each trial you can take as long as you want before starting.

So if you have to get up, return a phone call, stretch your legs that is the best time before starting the next trial.

The whole experiment should take no more than 15-25 minutes.

Please complete the whole experiment as I cannot use incomplete trial data.
