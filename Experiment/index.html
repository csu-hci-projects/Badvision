Badvision Keyboard Experiment

Hello and thank you for your interest! The experiment has concluded but is available for evaluation via this link if you are just curious what it looks like.

Note that answers are no longer recorded since the experiment is over.

About this experiment

The goal of this experiment is to compare how well you type using different types of keyboards and also to test out a protype typing hint system.

None of the data collected will be published or shared with any parties, this is only for a graded class project.

There are a total of 4 different trials, but the whole experiment should not take more than 15 minutes in total.

Technical details

All of the relevant information about the code used in this experiment and how to use it are in the "About" Github repo page, which is only accessible to CSU teaching faculty members (sorry!).
