Badvision Keyboard Experiment

Hello and thank you for participating in the Badvision keyboard experiment!

The goal of this experiment is to compare how well you type using different types of keyboards and also to test out a protype typing hint system.

Please note this requires a tablet and will not work with a mouse!

There are a total of 4 different trials, but the whole experiment should not take more than 15 minutes in total.

None of the data collected will be published or shared with any parties, this is only for a graded class project.
